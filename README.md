# One Piece Encyclopedia

A modern, interactive encyclopedia for One Piece, built with React, TypeScript, and Vite.

## Run locally

Prerequisites: Node.js

1. Install dependencies:
   `npm install`
2. Create a local environment file if needed and set `GEMINI_API_KEY`.
3. Start the app:
   `npm run dev`

## Build for production

```bash
npm run build
```
